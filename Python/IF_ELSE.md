# IF_ELSE en Python
La estructura 'if - else' sirve para tomar desiciones en el codigo si se cumple una condion se hace la accion,si no se cumplehace otra
tambien existe 'elif' para evaluar mas condiciones
## Ejemplo
nota = 65

if nota <= 90:
  print("Exelente")

elif nota <= 60:
  print("Aprobado")

else:
   print("Reprobado")
