# CICLO FOR en Python
El ciclo 'for' sirve para repetir una accion varias veces

## Codigo Ejemplo
'''Python

# Mostrar los numeros del 1 al 5

for numero in range(1,5):
  print("numero:",numero)

# Recorrer una lista de nombre

nombre = ["Romy","Alejo","Tama"]
for nombre in nombre:
  print("Hola", nombre)
