# Operaciones Basicas en Python

En python podemos hacer operaciones matematicas usando los simbolos +,-,*,/.

## Ejemplo 
 a = 10

 b = 3

suma = a + b  
# Resultado:13
resta = a - b 
# Resultado = 7
multiplicacion = a * b 
# Resultado 30 
division = a/b 
# Resultado 3.333

print("Suma:",suma)

print("resta:",resta)

print("Multiplicacion:",multiplicacion)

print("division:",division)
