## Diferencias Entre Python y Java

1. Java es estrictamente tipado: Debo declarar `int`, `String`, `boolean`. Y en Python no es necesario.
2. en java ocupo `private` y `public` para encapsular. En Python se usa `_` por covención.
3. El contructor en java se llama igual que la class `Personaje()`. En Python se usa `__init__`.
4. Para refrirme al objeto actual uso `this` en Java. En cambio en Python uso `self`.
5. Java debe copilarse antes de ejecutarse. En cambio Python interpreta directo.
