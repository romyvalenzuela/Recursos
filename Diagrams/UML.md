# ¿Que es UML y para que sirve?

UML significa lenguaje unificado de Modelado. 
Es un estandar gráfico que se usa para visualizar, diseñar y documentar sistema de software antes de escribir código.
Sirve como un plano para programadores.
Permite representar clases, objetos, casos de uso y secuencias de un programa usando diagramas.
En Programacion Orientada a Objetos es fundamental porque modela relacines como herencia, asociación y composición entre clases.
Ayuda a detectar errores de diseño temprano y facilita la comunicación entre desarrolladores y clientes.
Es la forma estandar de planificar software complejo.
